DOCUMENT

  • Document is representation the data loaded into browser window to the users
  • It comes under DOM(Documnet object model) both
  • It can control and manipulate the documnet nodes, such as element and the properties of the element

WINDOW

  • Window is the global object for the whole browser
  • It comes under BOM(Browser object model) and DOM(Documnet object model) both
  • It can control whole window(tab in modren browser), so it can close the window or bookmark the current document.
